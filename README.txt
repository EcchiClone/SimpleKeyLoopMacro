config.txt에 세팅을 작성하여 사용한다. 실행파일과 같은 폴더에 위치해야 한다.
----config.txt--------------------------------------------------------------------------------------------------------------
complete_5.mp3	첫 줄에는 효과음의 경로를 작성. 실행파일과 같은 폴더에 위치해야 한다.
3,*,shift+*,shift+r	둘째줄부터는 (delay 초, 시작단축키, 중단단축키, 반복할 스킬 키) 형식으로 작성한다.
3,ctrl+*,shift+*,f	쉼표","로 구분하여 총 네개의 인자를 작성. 단축키로는 쉼표와 +는 사용하지 않도록 주의.
3,alt+*,shift+*,ctrl+shift+b	원하는만큼 줄바꿈하여 추가 가능
--------------------------------------------------------------------------------------------------------------------------------
첫번째 줄인
[3,*,shift+*,shift+r]을 예로 들면
별표(*)를 눌러서 매크로를 시작하면 3초간격으로 Shift+r을 사용한다.
Shift+(*)을 눌러서 매크로를 중단한다.

delay의 경우, 명령줄의 실행에 걸리는 시간이 원인으로, 작성한 시간보다 약간의 시간이 더 소요됨에 주의. 소수점을 붙여 사용 가능하다.

네번째 인자인 스킬 단축키의 조합은 동시 세개 키까지 구현하였다. 즉
ctrl+shift+alt+a
처럼 네개 이상의 키는 동시에 사용할 수 없다.
